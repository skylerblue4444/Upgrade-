## 🚀 Getting Started

```bash
# Clone master repository
git clone https://github.com/skylerblue4444/Plz-I-m-done-shadowchat-.git
cd Plz-I-m-done-shadowchat-

# Navigate to main app
cd skycoin444_v10_live

# Install dependencies
pnpm install

# Start development
pnpm run dev

# Build for production
pnpm run build
```

## 🏗️ Architecture

- **Frontend:** React 19 + Tailwind CSS 4
- **Backend:** Express.js + tRPC
- **Database:** TiDB (MySQL-compatible)
- **Real-time:** Socket.IO
- **Payments:** Stripe API
- **AI:** OpenAI (HopeAI)
- **Storage:** AWS S3
